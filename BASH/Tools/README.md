# Outil pour server Linux

## Les scripts

### new-server.sh

Permet de configurer de facon static la configuration reseau d'un server  : 

- IP
- Netmask
- Gateway
- DNS

Permet de renommer le serveur

- Hostname

### Fonctionnement

Lancer le script 

`bash new-server.sh`
